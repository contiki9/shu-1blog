
# 概要
このリポジトリは週一ブログ会のサイト用です。


## URL
https://shu-1blog.com/
   
## Getting Started

```
$ git clone git@github.com:contiki9/shu-1blog.git
$ cd shu-1blog
$ npm install
$ npm start
```

## Commands

|Command|Description|
|:--|:--|
|`npm start`|Start development server|
|`npm run build`|Build files for publishing|
